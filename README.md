# Portfolio

A simplistic portfolio homepage built with Angular. Nothing fancy here yet—just a minimal landing page to showcase what I'm working on.

## About

This is my personal portfolio site. It's intentionally simple and clean, designed as a home page that I can expand on later as needed.

## Tech Stack

- **Angular** (v21)
- TypeScript
- HTML
- TailwindCSS

## The Angular Decision

Yeah, I know. I probably could've just thrown this together with vanilla JavaScript and HTML. Would've been faster, smaller, and honestly more appropriate for a simple static homepage.

But here we are with Angular. 🤷 Consider it overkill or consider it practice—either way, it works and it's here to stay (for now).

## Setup

```bash
# Install dependencies
npm install

# Run the development server
ng serve
```

Navigate to `http://localhost:4200/`. The app will automatically reload when you make changes.

## Build

```bash
ng build
```

The build artifacts will be stored in the `dist/` directory.
